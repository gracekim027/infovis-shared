# infovis-shared
## Setup Instructions

### Server

1. **Setup Python Environment:**
   ```bash
   pip install -r requirements.txt

2. **Run Server:**
   ```bash
   python run.py

### Client 

1. **Set OpenAI API Key:**
   - Add your OpenAI API key to the `.env` file:
     ```makefile
     OPEN_AI_API_KEY=your_openai_api_key_here
     ```

2. **Install Dependencies:**
   ```bash
   npm install

3. **Run Client:**
   ```bash
   npm start
